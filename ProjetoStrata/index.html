O que é o Projeto Strata?

O projeto foi criado com a intenção de praticar e memorizar todo o conteúdo que foi ensinado até o momento. Nessa página contém artigos, imagens, formulários, links, botões, tabelas, listas, textos e muito mais. Ele pode ser utilizado para consulta nos próximos projetos.
